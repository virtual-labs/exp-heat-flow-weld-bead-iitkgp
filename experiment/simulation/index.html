Instructions:

  1. Before starting the process read the instructions completely
  2. Press Start button to start the simulator.
  3. Select the type of speed 2 mm/s or 4 mm/s.
  4. Observe the Temperature Vs Time plot.
  5. Hover on the plot to see the temparature variation with time.
  6. Below the plot observe the Peak temperature values.
  7. Click on the Stop button to stop the simulation
